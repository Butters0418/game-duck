# 🦆 週末 H5 小遊戲 - 套鴨子

針對嵌入 APP WebView 需求，以原生 JavaScript 與 GSAP 開發的輕量 H5 遊戲

## 🎯 遊戲玩法

1. **蓄力投擲**：長按遊戲區域蓄力，放開後投擲圈圈
2. **瞄準距離**：透過蓄力條控制投擲距離
3. **得分規則**：
   - 近距離鴨子：5 分
   - 中距離鴨子：10 分
   - 遠距離鴨子：30 分
4. **獎勵機制**：達到 50 分以上可獲得額外獎勵
